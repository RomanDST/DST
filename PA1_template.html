Build a function to check if day a weekday or weekend

is_weekday <- function(d) { wd <- weekdays(d) ifelse (wd == “Saturday” | wd == “Sunday”, “weekend”, “weekday”) }

Apply the function

wx <- sapply(df_impute$date, is_weekday) df_impute$wk <- as.factor(wx) head(df_impute)

Build time series plot
